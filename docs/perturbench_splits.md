# PerturBench datasets & split availability

This note summarizes which splits are auto‑generated vs. manual, based on the official PerturBench README.

## Quick summary

- **Auto‑generated splits**: Srivatsan20 (Sciplex3) and Norman19.
- **Splits provided in Hugging Face repo (via accessors)**: all other datasets.
- **Manual split notebooks required**:
  - Frangieh21, Jiang24, OP3
  - McFalineFigueroa23 scaling experiments
  - Srivatsan20 imbalance experiments
